# ole-cli

> ⚽ CLI para la Agenda deportiva Olé

## Install

```
$ npm install --global https://github.com/sobrinojulian/ole-cli
```

## Usage

```
$ ole
# prints whole agenda
```


## Tip

Run it piped to `less` by adding `alias ole='ole | less'` to your `.zshrc`/`.bashrc`.
